**Що буде в коді**:
```javascript
async function saveSelectedEvents() {
  const selected = Array.from(document.querySelectorAll('.event-item.selected'))
    .map(el => el.dataset.eventId);
  
  const response = await fetch('/api/alarm-settings', {
    method: 'POST',
    body: JSON.stringify({ alarmEvents: selected })
  });
}
```
